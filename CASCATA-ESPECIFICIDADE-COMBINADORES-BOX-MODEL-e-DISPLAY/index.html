Regra da especificidade

A regra da especificidade consiste em dar mais peso para o elemento mais específico independente da ordem deles.

Seletor pela tag:

A especificidade do seletor pela tag é baixa (1), tendo menos peso para o site.

Seletor pela classe:

A especificidade do seletor pela classe já é maior, tendo peso 10 para o site.

Seletor pelo id:

A especificidade do seletor pelo id é ainda mais específico tendo peso 100 para o site.

Combinadores

A combinação de seletores consiste basicamente em combinar elementos com especificidades diferentes para obter estilos mais específicos.
